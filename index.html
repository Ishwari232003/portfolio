College Event Management System

A full-stack web app that centralizes all college events so students never miss seminars, hackathons, fests, or sports meets.
In college, students miss many events because information is scattered across WhatsApp groups, notice boards, and posters.
This system solves that problem by providing a single platform where all events are listed with dates, venue, description, and direct registration links.
Students can browse all upcoming events, filter by category, and click to view full event details.
Organizers can log in to a secure admin panel to add and delete events.
The system automatically hides past events and fetches all data from a real MySQL database through a REST API.

Tech:
Frontend — HTML, CSS, JavaScript
Backend — Java, Spring Boot
Database — MySQL
ORM — Spring Data JPA
Security — Spring Security

Career Match System

A full-stack web application that helps users choose the right career path based on their skills.
Many students feel confused while selecting a career because they don’t know which roles match their abilities, and most platforms provide generic or unclear suggestions.
This system solves that problem by allowing users to select their skills from 30+ options and instantly showing the most relevant career paths with accurate match percentage scores.
Each career recommendation includes useful details such as average salary and a step-by-step learning roadmap to guide users toward that role.
The system uses a custom-built skill matching algorithm that calculates real percentage scores from the database, ensuring that only meaningful and relevant matches are displayed instead of random suggestions.

Tech:
Frontend — HTML, CSS, JavaScript
Backend — Java, Spring Boot
Database — MySQL
ORM — Spring Data JPA
